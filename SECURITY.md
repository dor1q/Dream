# Security

## Секреты

Не коммитить:

- Discord bot token;
- JWT secrets;
- пароли;
- private keys;
- реальные пользовательские данные;
- `.env` и `.env.*`;
- игровые файлы и проприетарные ассеты.

Если секрет случайно попал в репозиторий:

1. Немедленно отзовите/смените секрет.
2. Удалите его из текущей версии.
3. Сообщите владельцу репозитория.

## Уязвимости

Пока проект приватно-рабочий, сообщения об уязвимостях лучше передавать владельцу напрямую. Не публикуйте exploit details в issues.

## Границы проекта

Документация и код в этом репозитории не должны включать инструкции по обходу защиты сторонних сервисов, распространению проприетарных файлов или получению несанкционированного доступа.
